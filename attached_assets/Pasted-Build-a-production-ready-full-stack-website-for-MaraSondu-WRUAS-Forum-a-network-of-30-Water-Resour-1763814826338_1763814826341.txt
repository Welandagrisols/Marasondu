2. API ENDPOINTS:

   Public APIs:
   - GET /api/projects - list projects (with filters)
   - GET /api/projects/:id - single project
   - GET /api/wruas - WRUA directory
   - GET /api/blog - blog posts (paginated)
   - GET /api/blog/:slug - single post
   - GET /api/stats - impact statistics
   - POST /api/contact - contact form submission
   - POST /api/newsletter - newsletter signup
   
   Admin APIs (auth required):
   - POST /api/admin/login
   - CRUD for projects, wruas, blog posts
   - POST /api/admin/upload - image upload
   - GET /api/admin/messages - contact form inbox
   - GET /api/admin/subscribers - newsletter list
